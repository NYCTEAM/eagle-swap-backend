# Legacy Swap API
Overview of Jupiter Legacy Swap API
[quoteRequest for a quote to be used in POST /swap](https://jupiter.mintlify.app/api-reference/swap/quote)

## quote

```
POST /swap
```

[swapRequest for a base64-encoded unsigned swap transaction based on the /quote response](https://jupiter.mintlify.app/api-reference/swap/swap)

## swap

```
/quote
```

[swap-instructionsRequest for swap instructions that you can use from the quote you get from /quote](https://jupiter.mintlify.app/api-reference/swap/swap-instructions)

## swap-instructions

```
/quote
```
